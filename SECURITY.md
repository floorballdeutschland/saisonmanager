# Security Policy

## Reporting a Vulnerability

Please do **not** report security vulnerabilities through public GitHub issues.

Instead, report them privately via one of these channels:

- Email: **it@floorball.de**
- GitHub: [private vulnerability reporting](https://github.com/floorballdeutschland/saisonmanager/security/advisories/new) on this repository

Please include steps to reproduce and, if possible, the affected view or component. We will confirm receipt within a few days and keep you informed about the fix.

## Supported Versions

The `main` branch reflects what runs in production at [saisonmanager.de](https://saisonmanager.de). Only the latest state receives security fixes; there are no maintained older release lines.

## Scope

This repository contains the Angular frontend. Vulnerabilities in the API belong to [saisonmanager-api](https://github.com/floorballdeutschland/saisonmanager-api) but can be reported through the same channels — we will route them internally.
